# Simplified POS Module Plan for Ko Hnit Aung Pharmacy

## Key Features to Implement:

1. **Smart Search**
   - 2-3 character partial matching
   - Search by name (English/Myanmar), SKU, shortCode
   - Instant results as user types

2. **Quick Buttons**
   - Top 10-20 selling products as large clickable buttons
   - Visual representation (color/image)
   - One-click add to cart

3. **Custom Price Override**
   - Inline price editing in cart
   - No admin permission required
   - Visual indicator when price is overridden

4. **Unit Conversion**
   - Dropdown to change unit per item in cart
   - Units: ကဒ်, လုံး, ဗူး, etc.
   - Automatic price recalculation if needed

5. **Traffic Light Stock Status**
   - Green: Good stock
   - Yellow: Low stock
   - Red: Out of stock/Expired

6. **Expiry Warnings**
   - Pop-up when trying to sell expired items
   - Visual warnings in product cards

7. **No Barcode Scanner**
   - Remove all scanner functionality
   - Focus on search and quick buttons
